# ML Python Summer 2026

Летний учебный репозиторий по Python, machine learning и ML engineering.

## Цель

Сформировать прочный инженерный фундамент для дальнейшего развития в сторону ML-инженерии.

## Структура

- `week01_python` — Python engineering, метрики, тесты
- `week02_math_ml` — математика для ML
- `week03_data` — NumPy, pandas, SQL, EDA
- `week04_classic_ml` — классическое машинное обучение
- `week05_pytorch` — deep learning и PyTorch
- `week06_mlops` — FastAPI, Docker, MLflow
- `projects` — итоговые проекты
- `notes` — конспекты
